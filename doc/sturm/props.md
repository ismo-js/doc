# Properties
## Meta
- **`IDX`** Index of the event
- **`LEN`** Length of the stream
- **`KEY`** Key of the event
- **`PRE`** Previous result
- **`CUR`** Circular to current event
- **`NXT`** Next event in the stream

## Value
- **`ERR`** Error
- **`RAW`** Numeric value
- **`STR`** String value
- **`FUN`** Function
